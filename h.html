Introduction to HTML Tags

HTML (HyperText Markup Language) is the standard language for creating web pages. Tags in HTML are used to define the structure and content of a webpage. Tags are the building blocks of HTML and are enclosed in angle brackets like <tag>.

Structure of an HTML Tag

Each HTML tag has two parts:

  • Opening Tag: Marks the beginning of the element (e.g., <h1>).
  • Closing Tag: Marks the end of the element (e.g., </h1>).
  • Content: The information enclosed within the tags (e.g., the text or image that is displayed on the webpage).

Common HTML Tags

HTML tags are used to structure and format content on a webpage. Here are some of the most common HTML tags and their uses:

1. Structural Tags

  • <html> - The root element of an HTML document.
  • <head> - Contains metadata about the HTML document, like the title and character encoding.
  • <body> - Contains the main content of the document, which is visible on the page.
